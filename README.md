## 📌 About The Project

**ShareBoard** is a full-stack web application that allows users to share **text notes**, **audio recordings**, and **images** using a simple 6-character code. No login required! Just create, get a code, and share with anyone.

### ✨ Key Features

- 📝 **Text Notes** - Write and share text messages
- 🎤 **Audio Recording** - Record voice messages directly in browser
- 📸 **Image Sharing** - Upload and share images (JPG, PNG, GIF)
- 🔑 **6-Character Codes** - Unique codes for every share
- ⏰ **24-Hour Auto-Expiry** - Notes automatically delete after 24 hours
- 👁️ **View Counter** - Track how many times your note was viewed
- 📋 **Copy to Clipboard** - One-click code copying
- 📱 **Fully Responsive** - Works perfectly on all devices
- 🎨 **Beautiful Animations** - Smooth transitions with Framer Motion
- 🌐 **No Login Required** - Completely anonymous
